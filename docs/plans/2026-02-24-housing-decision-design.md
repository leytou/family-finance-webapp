# 买房 vs 不买房 财务规划对比 — 设计文档

日期：2026-02-24

## 目标

在现有五年财务规划表中增加"买房 vs 不买房"场景对比，帮助做出购房决策。

## 方案概述（方案 C：参数共享 + 月度表独立）

- 总览表新增"买房假设"参数区域
- 新增"月度现金流（买房）"工作表
- 总览表五年收支总览扩展为并排对比

## 一、总览表 — 买房假设参数区域

在现有假设区域（row 4-14）下方新增：

### 新增行布局

| 行号 | 内容 |
|------|------|
| 16   | 章节标题 "🏠 买房假设" |
| 17   | 子标题行（与 row 5 格式一致） |
| 18   | 计划买房时间 \| 202801 \| YYYYMM格式 |
| 19   | 总房价 \| 1500000 \| 元 |
| 20   | 首付比例 \| 0.3 \| 30% |
| 21   | 贷款年限 \| 30 \| 年 |
| 22   | 贷款年利率 \| 0.031 \| 商贷参考利率 |

### 计算行（公式自动计算，不可编辑）

| 行号 | 内容 |
|------|------|
| 23   | 首付金额 = 总房价 × 首付比例 |
| 24   | 贷款总额 = 总房价 - 首付金额 |
| 25   | 月供金额 = PMT(月利率, 期数, 贷款总额) |

所有买房参数支持现有的时间变更列（D-M），与基本假设保持一致。

五年收支总览从 row 27 开始（原 row 15 下移）。

## 二、月度现金流（买房）工作表

### 列结构（18列）

| 列 | 内容 | 说明 |
|----|------|------|
| A  | 序号 | 同不买房 |
| B  | 年份 | 同不买房 |
| C  | 月份 | 同不买房 |
| D  | 月收入 | 引用总览表同一假设 |
| E  | 年终奖 | 引用总览表同一假设 |
| F  | 理财收益 | 基于本表累计储蓄计算 |
| G  | 总收入 | =D+E+F |
| H  | 房租 | 与不买房表完全相同（不自动清零） |
| I  | 日常开销 | 引用总览表同一假设 |
| J  | 孩子开销 | 引用总览表同一假设 |
| K  | 特殊支出 | 同不买房（结婚、孕产、旅游） |
| L  | 特殊说明 | 同不买房 |
| M  | 首付支出 | 仅买房当月 = 总房价 × 首付比例 |
| N  | 房贷月供 | 买房次月起，等额本息月供 |
| O  | 总支出 | =SUM(H:K)+M+N |
| P  | 当月净储蓄 | =G-O |
| Q  | 累计储蓄 | 上月累计+当月净储蓄 |
| R  | 房贷剩余 | 贷款余额递减 |

### 关键公式逻辑

- **首付支出（M列）**：`=IF(年月=买房时间, 总房价×首付比例, 0)`
- **房贷月供（N列）**：`=IF(年月>买房时间, PMT月供, 0)`
- **房贷剩余（R列）**：初始=贷款总额，每月减去本金部分
- **房租（H列）**：与不买房表完全相同，不做修改
- **理财收益（F列）**：基于本表自身的累计储蓄计算

### 与不买房表的差异

仅增加 M（首付）、N（月供）、R（房贷剩余）三列，其余列公式引用相同的总览表假设参数。

## 三、总览表 — 五年收支总览扩展

### 布局

```
A列      | B-F列（不买房方案） | G列（间隔） | H-L列（买房方案）
项目     | 2026  2027 ... 2030 |             | 2026  2027 ... 2030
```

### 行结构

买房方案的行结构与不买房方案相同，额外增加：
- 首付支出行
- 房贷月供行

两个方案的数据分别引用各自的月度现金流表。

### 标题标注

- B-F 列标题行标注"📋 不买房方案"
- H-L 列标题行标注"🏠 买房方案"

## 四、技术实现要点

1. **代码复用**：月度现金流生成逻辑提取为函数，买房表和不买房表共用
2. **行号管理**：买房假设区域插入后，五年收支总览行号整体下移
3. **等额本息公式**：Excel PMT 函数 `=-PMT(月利率, 总期数, 贷款总额)`
4. **时间判断**：买房时间用 YYYYMM 格式，与现有时间变更机制一致
5. **样式一致**：新增内容沿用现有配色方案（简约现代风）
