# 表格样式优化设计：简约现代风配色

## 目标
将财务规划表格从传统 Office 蓝色系升级为简约现代风，解决颜色刺眼/不协调的问题。

## 方案：全面换色（仅改颜色常量，不动公式和布局）

### 颜色映射

| 用途 | 当前值 | 新值 | 说明 |
|------|--------|------|------|
| BLUE（输入值字体） | 0000FF | 2B6CB0 | 柔和蓝 |
| BLACK（公式字体） | 000000 | 2D3748 | 深灰蓝 |
| GREEN（链接公式字体） | 008000 | 2F855A | 柔和绿 |
| DARK_BG（标题背景） | 1F4E79 | 2D3748 | 深灰蓝 |
| MED_BG（章节标题背景） | 2E75B6 | 4A90D9 | 柔和蓝 |
| LIGHT_BG（参数名背景） | DEEAF1 | EDF2F7 | 浅灰蓝 |
| YELLOW_BG（输入格背景） | FFFF00 | EBF4FF | 淡蓝（最大改善点） |
| GREEN_BG（收入/年终奖） | E2EFDA | E6F4EA | 淡绿 |
| RED_BG（支出小计） | FCE4D6 | FDE8E8 | 淡红 |
| AMBER_BG（特殊支出/结余） | FFF2CC | FEF3C7 | 淡琥珀 |
| GRAY_BG（偶数行/备注） | F2F2F2 | F7FAFC | 极浅灰蓝 |
| ORANGE_BG | FCE4D6 | FDE8E8 | 与 RED_BG 统一 |

### 硬编码颜色同步更新

| 位置 | 当前值 | 新值 |
|------|--------|------|
| 子标题行背景 4472C4 | → 4A90D9（与 MED_BG 统一） |
| 备注字体 595959 | → 718096（柔和灰） |
| 章节标题字体 1F4E79 | → 2D3748（与 DARK_BG 统一） |
| SECT_HDR_BG D6E4F0 | → E2E8F0（柔和灰蓝） |

### 不变的部分
- 所有公式逻辑
- 字体大小层次（16/12/11/10/9）
- 行高、列宽
- 边框样式
- 对齐方式
