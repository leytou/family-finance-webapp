# 公积金专区列编辑高亮与右键清除菜单 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让月度表的「房贷月供」「公积金缴存」两列，编辑后显示浅蓝底，并支持与动态列一致的右键菜单（「同步到下方每年此月」恒置灰、「清除该值」、「清除下方编辑值」）。

**Architecture:** 仅改 `MonthlyTable.vue`。给两列各配一个内部 sentinel 标识，接进现有的右键菜单统一逻辑（`openContextMenu` / `contextMenuItems` / 清除系列函数），按 sentinel 分支读写 `fund.<field>.entries`。不改 store、不改计算、不改菜单组件。

**Tech Stack:** Vue 3 (Composition API + `<script setup>` + TS)、UnoCSS（`bg-brand-50`）、Vitest + @vue/test-utils。

**对应设计文档：** `docs/superpowers/specs/2026-06-15-公积金专区列编辑高亮与右键菜单-design.md`

```ts
// 房贷月供 / 公积金缴存列在右键菜单逻辑中的特殊列标识
const FUND_MORTGAGE_COLUMN_ID = '__fund_mortgage__'
const FUND_CONTRIBUTION_COLUMN_ID = '__fund_contribution__'

// 标识 → fund 字段名；非专区固定列返回 null
function fundFieldFromColumnId(columnId: string): 'mortgage' | 'contribution' | null {
  if (columnId === FUND_MORTGAGE_COLUMN_ID) return 'mortgage'
  if (columnId === FUND_CONTRIBUTION_COLUMN_ID) return 'contribution'
  return null
}
```

```ts
function isCurrentCellEdited(columnId: string, month: number): boolean {
```
