# 收入 / 支出 / 结余 口径重分类 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把理财收益与公积金提取归入「收入」，公积金月冲改为抵扣房贷（支出只记自己实际掏的「存款补扣」），让「结余 = 收入 − 支出」直接成立；年度表新增「年收入」「年支出」两行。最终存款数字不变。

**Architecture:** 改 `calculate()` 的 `monthlyIncome/monthlyExpense/monthlyBalance` 计算口径（数值与旧口径等价，累计储蓄不变）；同步更新月度/年度公式弹窗文案；年度表 computed 增 `yearIncome/yearExpense` 并新增两行。图表、多方案对比、按年聚合均直接读取这三列，口径一改自动对齐，无需改动其逻辑。

**Tech Stack:** Vue 3 (Composition API + TS) + Vitest。单次运行测试用 `npx vitest run <file>`。

参考设计：`docs/superpowers/specs/2026-06-14-收入结余口径-design.md`
