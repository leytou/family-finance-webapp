# 家庭财务规划网页应用 — 需求设计

## 1. 产品定位

- 私人财务规划工具，仅供自己使用
- 从当前月起规划未来 5 年的现金流
- 浏览器本地运行，数据自动保存在本地

## 2. 核心模型：统一现金流项目

所有收入和支出都用同一种结构表达：

- **名称**：自由文字（如"工资"、"房租"、"结婚"）
- **类型**：收入 / 支出
- **金额段**：一个或多个分段，每段 = 金额 + 时间范围
  - 时间范围默认：起始 = 当前月，结束 = 规划末
  - 同期只生效一个值（后填段覆盖先填段）
- **时间选择交互**：网格滑动（每年一行 × 12 格，从当前月起共 5 年）

## 3. 系统参数

- **当前总储蓄**：绑定当前月份（规划起始月），是滚动计算的起点，本质上是第一个锚点
- **年化理财收益率**：默认 2.5%

## 4. 月度锚点

- 可在任意月份设置"实际总储蓄"锚点值
- 当前总储蓄 = 第一个锚点（当前月）
- 锚点月的累计储蓄直接等于锚点值（覆盖预估累积）
- 锚点之后的月份从锚点值重新滚动计算

## 5. 计算逻辑

- 每月收入合计 = 该月所有收入项之和
- 每月支出合计 = 该月所有支出项之和
- 每月理财收益 = 上月末累计储蓄 × 年利率 / 12
- 每月净储蓄 = 收入合计 − 支出合计 + 理财收益
- 每月累计储蓄 = 上月末累计储蓄 + 当月净储蓄（遇锚点则直接取锚点值）

## 6. 展示视图

- **年度汇总表**：年初储蓄、全年收入/支出逐项列出、年度结余、年末储蓄
- **月度明细表**：每月各项收支、理财收益、净储蓄、累计储蓄
- **计算公式透视**：点击任意数字可展开查看计算来源

## 7. 交互方式

- 参数面板：集中展示所有现金流项目和系统参数
- 实时联动：修改任意参数，表格即时重算
- 网格滑动选时间：按住鼠标在月份网格上滑动来选择时间段

## 8. 显示原则

- 数据显示密度高：在有限空间内尽可能多地展示财务数据
- 表格紧凑：行高列宽精简，减少留白，一屏尽量看全
- 数字为主：避免冗余文字描述，让数字本身说话

## 9. 数据持久化

- 浏览器 localStorage 自动保存，打开即恢复
