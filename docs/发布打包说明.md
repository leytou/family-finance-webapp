# 发布打包说明（给非前端同学看）

## 一句话
每次想把最新功能发给朋友，只需跑一条命令：`npm run build`，然后把 `dist/index.html` 发给朋友即可。

---

## 为什么是单文件
项目配置了「打包成单个网页文件」。所以每次打包后，`dist` 文件夹里**只有一个 `index.html`**，它包含了全部功能（界面、计算、图表），双击就能直接用。朋友拿到它不需要任何其他东西。

---

## 打包三步走

1. 在项目文件夹（`plan`）里打开命令行
2. 敲这一条命令，回车，等几秒：
   ```
   npm run build
   ```
3. 看到 `✓ built in ...` 就表示成功。成品就在：
   > `plan/dist/index.html`

---

## 命令在哪敲（任选一种）

- **VS Code 里**：打开 `plan` 文件夹 → 顶部菜单「终端」→「新建终端」→ 在弹出的窗口里敲命令
- **文件夹里直接开**：打开 `plan` 文件夹 → 点上方地址栏 → 输入 `cmd` 回车 → 弹出的黑窗口里敲命令
- **正在用 Claude Code 时**：直接说「帮我打包一下」，我来跑

---

## 发给朋友

1. 把 `dist/index.html` 复制出来（可以改个好认的名字，如 `家庭财务规划.html`）
2. **压缩成 zip 再发**（避免微信等工具拦截网页文件）
3. 朋友解压后，双击 `.html` 用浏览器打开即可

---

## 注意事项

- 只要**没换电脑、没删 `node_modules`**，以后每次直接 `npm run build` 就行。
- 如果**换了电脑**或**重新拉了代码**，第一次要先跑 `npm install` 装依赖（只需一次），之后就能正常打包。
- 打包前建议先 `npm run dev` 本地跑一下，确认没问题再打包发布。
